# Support

Use GitHub Discussions for questions, deployment help, and design conversation:

- FreeSWITCH deployment and SIP/NAT questions
- IVR builder and runtime behavior
- MCP / AI-agent usage
- n8n and workflow automation
- operational guidance and release planning

Use GitHub Issues for scoped, actionable work:

- reproducible bugs
- implementation slices
- documentation gaps
- security hardening tasks that do not disclose vulnerabilities

Do not open public issues for suspected vulnerabilities. Use private security
reporting as described in `SECURITY.md`.
